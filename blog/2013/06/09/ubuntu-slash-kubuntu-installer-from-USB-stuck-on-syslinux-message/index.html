Ubuntu/Kubuntu Installer From USB Stuck on SYSLINUX Message

Jun 9th, 2013

Last week I was having problems installing Kubuntu 13.04 on my laptop. After copying the ISO to a USB stick, the installer won’t show up. I tried with Universal USB installer and Unetbootin, but only got a message like this:

SYSLINUX 4.03 2010-10-22 EDD Copyright (c) 1994-2010, H. Peter Anvin et al

The problem was that the stick was formatted on FAT32, using the Restore device defaults button on the Windows 7 format tool changed to FAT and the installer ran fine.
